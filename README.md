# GE-Switch

基于 RTL8367RB 的 5 口千兆交换机。

体积很小巧，PCB 大小 54x81mm，4 层板；主要元件采用 0603 封装，方便手工焊接；网口使用 HanRun 的 HR911130A 千兆网口，自带隔离变压器；Type-C 供电。
